# Stepper Motor Control

Stepper Motor Control에 대한 template code이다.

Stepper motor는 두 가지 방식으로 제어할 수 있다.

- Acceleration 기반 제어
- Velocity 기반 제어

또한 stepper motor는 세 가지 방식으로 상을 잡아줄 수 있다.

- 1상 여자 방식
- 2상 여자 방식
- 1, 2상 여자 방식
